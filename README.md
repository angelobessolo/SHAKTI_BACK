# Ejecutar En Desarrollo SHAKTI

1. Clonar repositorio
2. Ejecuta

```
npm install 
```

3. Tener nest cli instalado

```
npm i -g @nestjs/cli  
```

4. Levantar base de datos

```
docker-compose up -d
```

5. Levantar backend npm run start:dev

# Stack Utilizado

* Mongo DB
* NestJS
